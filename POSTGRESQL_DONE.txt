🚀 ÉTAPES SUIVANTES:

1️⃣ Installer PostgreSQL
   📥 Télécharger depuis: https://www.postgresql.org/download/windows/
   
2️⃣ Configuration automatique (Windows)
   🔄 Double-cliquez sur: SETUP_POSTGRESQL.bat
   
   OU
   
   Configuration manuelle:
   💻 psql -U postgres -c "CREATE DATABASE laravel_api;"

3️⃣ Vérifier la connexion
   ✓ php artisan db:check

4️⃣ Exécuter les migrations
   ✓ php artisan migrate:fresh --seed

5️⃣ Démarrer l'API
   ✓ php artisan serve --port=8001
   
6️⃣ Tester les endpoints
   ✓ Importer postman_collection.json dans Postman
   ✓ OU utiliser test-alerts.php
