🔗 ENDPOINTS PRINCIPALES:

AUTHENTIFICATION:
  POST   /auth/register              - Créer utilisateur
  POST   /auth/login                 - Se connecter
  POST   /auth/logout                - Se déconnecter
  GET    /auth/me                    - Profil actuel

PROFIL:
  GET    /profile                    - Récupérer profil
  PUT    /profile                    - Modifier profil
  PUT    /profile/change-password    - Changer mot de passe
  PUT    /profile/notification-preferences - Modifier préférences

HISTORIQUE:
  GET    /activity-logs              - Tous les logs
  GET    /activity-logs/user/{id}    - Logs d'un utilisateur
  GET    /activity-logs/{id}         - Détail d'un log

GESTION (Admin):
  GET    /users                      - Liste utilisateurs
  GET    /users/{id}                 - Détail utilisateur
  POST   /users/{id}/assign-role     - Assigner rôle
  DELETE /users/{id}                 - Supprimer utilisateur
  GET    /roles                      - Liste rôles
  POST   /roles                      - Créer rôle
  DELETE /roles/{id}                 - Supprimer rôle

🆘 SUPPORT:

Problème de démarrage?
  → Vérifiez que PHP 8.3+ est installé
  → Lancez: php artisan key:generate
  → Lancez: php artisan migrate:fresh --seed

BD non accessible?
  → Vérifiez le fichier .env (DB_*)
  → Mysql ou SQLite: driver doit être configuré

Tokens invalides?
  → Vérifiez que vous incluez "Authorization: Bearer {token}"
  → Le token doit être sans guillemets
